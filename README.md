# Chess-Server
-----------------
1) Загрузить репозиторий :
git clone https://github.com/OlegRabushko/Chess-Server.git
----------
2) Установить зависимости :
npm install
------------
3) Запустить сервер :
npm start
-----------
4) Открыть ещё один терминал и запустить webSocket: 
npm run socket
------------
5) Для игры по сети откройте 2ой браузер.
